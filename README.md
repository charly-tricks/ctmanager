# CTManager 🛡️
### Gestor Todo en Uno - by CHARLY_TRICKS

## Instalación
```bash
wget https://raw.githubusercontent.com/charly-tricks/ctmanager/main/ctmanager.sh && chmod +x ctmanager.sh && ./ctmanager.sh
```

## Módulos incluidos
- 🚀 Hysteria V1/V2
- 🌐 WebSocket Proxy (WS/WSS)
- 🐍 SOCKS Python (Injector)
- 📡 BadVPN / UDP-Custom
- 🌍 SlowDNS / DNSTT
- 🔐 Psiphon
- 🌊 Brook WS/WSS
- 🔑 Dropbear Multi-puerto
- 🔒 Stunnel SSL
- 🛡️ OpenVPN

## Sistemas soportados
- Ubuntu / Debian
- CentOS / AlmaLinux / Rocky Linux
- Fedora / RHEL
- Arch / Manjaro

## Comando de gestión
Después de instalar, usá el comando:
```bash
ctmanager
```
